13. REGRAS IMPORTANTES DE IMPLEMENTAÇÃO
================================================================================

13.1 RESPONSIVIDADE

- SEMPRE usar classes responsivas: lg:ml-64, lg:hidden, etc
- Mobile-first: base style sem prefixo, desktop com lg:
- Testar em telas: mobile (375px), tablet (768px), desktop (1024px+)

13.2 CORES

- SEMPRE usar as cores definidas no design system
- Verde (#22c55e) é a cor PRIMÁRIA
- Usar gray-50 para background geral
- white para cards e sidebar

13.3 ESPAÇAMENTOS

- Manter consistência: p-3 mobile, p-6 desktop
- space-y-6 entre seções principais
- space-y-4 entre cards
- gap-4 em grids

13.4 FONTES

- Usar font-medium para títulos secundários
- font-semibold para títulos importantes
- font-bold apenas para H1/H2
- text-sm para body, text-xs para detalhes

13.5 INTERAÇÕES

- SEMPRE adicionar hover states
- Transições de 300ms
- Feedback visual em todas ações
- Loading states obrigatórios

13.6 ACESSIBILIDADE

- Titles nos botões quando collapsed
- Alt text nas imagens
- ARIA labels onde necessário
- Focus states visíveis
- Contraste adequado

13.7 PERFORMANCE

- Lazy loading de componentes pesados
- Imagens otimizadas
- Server components quando possível
- Debounce em buscas
