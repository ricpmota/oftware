# Etapa 11 — Apêndice: glossário e links

## 11.1 Glossário

| Termo | Significado |
|-------|-------------|
| **Token** | Nome semântico (`brand-primary`) ligado a um valor de cor |
| **Tailwind** | Framework CSS utilitário (`bg-green-600`) |
| **Viewport** | Largura da janela do navegador |
| **`md:` / `lg:`** | Prefixos Tailwind = estilo a partir de breakpoint |
| **PR** | Pull request (mudança no código) |
| **WCAG** | Diretrizes de acessibilidade web |

## 11.2 Breakpoints Tailwind (padrão)

| Prefixo | Mínimo |
|---------|--------|
| sm | 640px |
| md | 768px |
| lg | 1024px |
| xl | 1280px |
